# 🐍 Python Mini Projects

A curated collection of beginner-to-intermediate Python projects focused on data processing, analysis, visualization, and problem-solving.

This repository is built as a learning + portfolio space to showcase practical Python skills and continuous improvement through hands-on projects.

---

## 📁 Projects Included

### 📊 1. Data File Processor & Expense Analysis
A complete data processing pipeline that:
- Reads raw CSV expense data
- Cleans and preprocesses the data
- Performs analysis and generates insights
- Produces visual charts (daily spending, hourly trends, category comparison, weekdays vs weekends)
- Generates a structured report and HTML documentation

Tech used: Python, Pandas, Matplotlib, Jupyter, HTML, Git

📂 Folder: Data_File_Processor/

---

### 🎵 2. Spotify Data Analysis
A data analysis project that explores:
- Most streamed songs
- Top artists by streams
- Popularity distribution
- Data cleaning, aggregation, and visualization

Tech used: Python, Pandas, Matplotlib, Jupyter

📂 Folder: spotify_analysis/

---

### 🧮 3. Simple Calculator
A basic Python command-line calculator supporting:
- Addition, subtraction, multiplication, division
- Power, remainder, square root

📂 Folder: PROJECTS_PY/

---

### 🧩 4. Sudoku Solver
A Sudoku solver built using recursion and backtracking.

Features:
- Solves valid 9×9 Sudoku puzzles
- Row, column, and 3×3 box validation
- Recursive backtracking algorithm
- Modular project structure

Concepts learned: Recursion, Backtracking, 2D Arrays, Constraint Satisfaction, Algorithm Design

Tech used: Python

📂 Folder: sudoku_solver/
