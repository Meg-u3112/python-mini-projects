# Python Mini Projects 🐍

A collection of beginner-friendly Python projects to practice programming concepts.  
Currently included:

## 📂 Projects

### 1️⃣ Simple Calculator 🧮
A command-line calculator that supports:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Power (^)
- Remainder (%)
- Square root (√)
---

### 2️⃣ Spotify Data Analysis 🎵

An end-to-end data analysis project using Spotify song data.

**Highlights:**
- Data cleaning and preprocessing
- Duplicate and null value handling
- Stream count analysis and artist ranking
- Project documentation using HTML

📂 Folder: `spotify_analysis/`  
📄 Documentation: `spotify_analysis/docs/index.html`
