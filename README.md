# 🐍 Python Mini Projects

A curated collection of beginner-to-intermediate Python projects focused on data processing, analysis, and visualization.  
This repository is built as a **learning + portfolio** space to showcase practical Python skills.

---

## 📁 Projects Included

### 📊 1. Data File Processor & Expense Analysis
A complete data processing pipeline that:
- Reads raw CSV expense data
- Cleans and preprocesses the data
- Performs analysis and generates insights
- Produces visual charts (daily spending, hourly trends, category comparison, weekdays vs weekends)
- Generates a structured report and HTML documentation

**Tech used:** Python, Pandas, Matplotlib, Jupyter, HTML, Git

📂 Folder: `Data_File_Processor/`

---

### 🎵 2. Spotify Data Analysis
A data analysis project that explores:
- Most streamed songs
- Top artists by streams
- Popularity distribution
- Data cleaning, aggregation, and visualization

**Tech used:** Python, Pandas, Matplotlib, Jupyter

📂 Folder: `spotify_analysis/`

---

### 🧮 3. Simple Calculator
A basic Python command-line calculator supporting:
- Addition, subtraction, multiplication, division
- Power, remainder, square root

📂 Folder: `PROJECTS_PY/`
