# Spotify Data Analysis 🎵

## Overview
This project  focuses on analyzing Spotify song data to identify the most streamed songs and top artists using Python and data analysis techniques.

## Objectives
- Clean and preprocess Spotify data
- Remove null and duplicate entries
- Analyze song popularity using stream counts
- Identify top-performing artists
- Visualize key insights

## Project Structure
spotify_youtube_analysis/
├── data/             # Raw Spotify dataset
├── notebooks/        # Jupyter Notebook with analysis
├── analysis/         # Cleaned data and insights
├── visualizations/   # Charts and plots
├── reports/          # HTML/PDF reports
└── web/              # Static HTML summary

## Tools & Technologies
- Python
- pandas
- matplotlib
- Jupyter Notebook

## Output
- Jupyter Notebook with full analysis
- Cleaned dataset
- Visualizations of top songs and artists
- HTML report for presentation
