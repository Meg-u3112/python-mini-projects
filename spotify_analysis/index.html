🎵 Spotify Data Analysis

📌 Project Overview

This project focuses on analyzing Spotify song data to identify the most streamed songs and top-performing artists using Python and data analysis techniques.

🎯 Objectives

  • Clean and preprocess raw Spotify data
  • Remove null and duplicate records
  • Analyze song popularity using stream counts
  • Identify top artists based on total streams
  • Present insights in a clear and structured manner

⚙️ Project Workflow

  1. Data Collection: Spotify dataset loaded from CSV files
  2. Data Cleaning: Null values and duplicate songs removed
  3. Data Validation: Stream counts converted to numeric format
  4. Analysis: Ranking songs and aggregating artist streams
  5. Visualization: Key insights visualized using charts

🧠 Tools & Technologies

  • Python
  • pandas
  • matplotlib
  • Jupyter Notebook
  • Git & GitHub

✅ Key Outcomes

  • Identified the most streamed songs on Spotify
  • Determined top artists based on total streams
  • Built a clean, reproducible data analysis pipeline
